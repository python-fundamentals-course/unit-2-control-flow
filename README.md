# Unit 2: Control Flow

> **Python Fundamentals**

if/elif/else, for loops, while loops, and the three patterns that cover most real data tasks.

## New here?

If this is your first time opening a unit repo, start with
[**GETTING_STARTED.md**](GETTING_STARTED.md) — it covers opening this
Codespace, using Jupyter notebooks, doing the exercises, and checking your
work. You don't need to install anything.

## Learning objectives

By the end of this unit you will be able to:

- Write if/elif/else blocks for business decision logic
- Apply for loops to filter, transform, and aggregate data
- Decompose a business problem into control flow before writing code
